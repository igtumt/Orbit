# ☁️ Orbit OS
**AI-Powered, WebAssembly-Based Cloud Operating System**

Orbit OS instantly analyzes your Rust code from GitHub, autonomously generates a UI (User Interface), and allows you to test it directly in your browser without downloading anything!



## ✨ Features
- **🤖 AI (Gemini) Integration:** Understands the logic of your code and dynamically draws the appropriate input boxes and buttons.
- **🚀 WebAssembly (WASM):** Your code is securely compiled inside a Docker container and delivered to your browser in milliseconds.
- **🖥️ Desktop Experience:** Features a window manager that allows you to drag, drop, close, and run multiple applications side-by-side.
